# Contributing to Waterfall Tools

Welcome! All contributions are welcome and greatly appreciated.

By contributing to this repository, you assert that you have the rights to the code you are contributing and you agree to be bound by the project's Apache 2.0 license for any code that you contribute.

## AI-Assisted Development

This project is actively designed to be used in AI-assisted development workflows. 
When making structural changes, API modifications, or adding significant functionality:
- Please ensure that you also include relevant updates to the `AGENTS.md` file. This acts as the project's long-term memory to help future AI agents understand the context, boundaries, and rationale behind your changes.
- Remember to update any relevant project documentation inside the `Docs/` directory or `README.md` to reflect the new state.

Thank you for helping improve Waterfall Tools!
